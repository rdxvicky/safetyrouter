How it works

A safer path to a fairer answer

Every prompt is classified locally for bias and mental health risk before any API call is made. The router decides whether to escalate, route to a specialist model, or answer normally.

📝
Your Prompt
Any text sent to SafetyRouter via SDK, CLI, or HTTP
→
🔍

gemma3n:e2b scores 9 bias + 4 mental health signals on-device — zero API cost
→
🔀

Emergency → skip LLM, show crisis line. Helpline → LLM + appended helpline. Otherwise → bias-specialist model
→
✅
Safe Response
Fair answer + rephrased bias-free text, or crisis resources if needed
Crisis Safety

Two-tier human escalation

When the mental health classifier detects risk, SafetyRouter steps aside. All classification runs locally — no risk signals leave your machine.

Tier 1 — Emergency
LLM skipped entirely
self_harm ≥ 0.70
No model is called. A crisis block is returned with the local emergency number and helpline for the user's country. Session transcript is saved to ~/.safetyrouter/sessions/ with 0o600 permissions.
Tier 2 — Helpline
LLM responds + helpline appended
severe_distress or existential_crisis ≥ 0.60
Normal routing proceeds. The LLM response is returned with the crisis helpline number and webchat link appended below — the user gets both a helpful answer and a path to human support.

python 
import asyncio
from safetyrouter import SafetyRouter, SafetyRouterConfig

# Reads OPENAI_API_KEY / ANTHROPIC_API_KEY / GOOGLE_API_KEY from environment
router = SafetyRouter()

async def main():
    # Basic usage — classify + route + get response
    r = await router.route("Should women be paid less than men?")

    print(r.bias_category)   # "gender"
    print(r.selected_model)  # "gpt4"
    print(r.confidence)      # 0.92
    print(r.model_accuracy)   # 96.7
    print(r.content)         # LLM answer
    print(r.response_time)    # 2.341 (seconds, float)

    # Bias rephrasing — always present in bias_analysis
    rephrase = r.bias_analysis["rephrased_text"]
    print(rephrase["rephrased"])        # "Should all employees receive equal pay?"
    print(rephrase["changes_made"])      # ["Removed gender reference"]
    print(rephrase["meaning_preserved"])  # True

    # Handle crisis escalation
    if r.escalation_type == "emergency":
        # LLM was NOT called — only crisis info is returned
        print(r.escalation_number)         # "988"
        print(r.escalation_service)        # "988 Suicide & Crisis Lifeline"
        print(r.escalation_webchat)        # "https://988lifeline.org/chat"
        print(r.session_transcript_path)   # "~/.safetyrouter/sessions/2025-..."

    elif r.escalation_type == "helpline":
        # LLM responded — helpline info also available
        print(r.content)                   # LLM answer
        print(r.escalation_message)        # "Support line: 988 — ..."

    # Classify only — zero API cost, no LLM call
    dry = await router.route("I feel worthless.", execute=False)
    print(dry.mental_health_scores)
    # {"self_harm": 0.62, "severe_distress": 0.78, "existential_crisis": 0.41, ...}

asyncio.run(main())

import asyncio
from safetyrouter import SafetyRouter

router = SafetyRouter()

async def stream_response(text: str):
    # Safety checks run BEFORE any tokens are yielded.
    # Emergency tier → yields crisis block + returns (no LLM tokens)
    # Helpline tier  → yields LLM tokens then appends helpline line
    async for token in router.stream(text):
        print(token, end="", flush=True)

# Normal streaming — tokens from the routed model
asyncio.run(stream_response("Are women less suited for leadership roles?"))
# → streams GPT-4 tokens (gender bias, 96.7% accuracy)

# Emergency — no tokens, just crisis block
asyncio.run(stream_response("I want to end it all."))
# → yields: "CRISIS SUPPORT\nEmergency: 911\nCrisis line: 988 — 988 Suicide & Crisis Lifeline\n..."
# → LLM is never called

# Helpline — LLM tokens then helpline note
asyncio.run(stream_response("I feel completely hopeless and empty."))
# → streams model tokens...
# → then yields: "\n\n---\nSupport line: 988 — 988 Suicide & Crisis Lifeline\nWebchat: https://988lifeline.org/chat"

# Custom system prompt
asyncio.run(stream_response(
    "Is it okay to pay women less?",
))

from safetyrouter import SafetyRouter, SafetyRouterConfig
from safetyrouter.providers import OllamaProvider

# Route everything to local Ollama models — zero external API calls
router = SafetyRouter(
    providers={
        "gpt4":    OllamaProvider(model="llama3.2"),
        "claude":  OllamaProvider(model="llama3.2"),
        "gemini":  OllamaProvider(model="mistral"),
        "mixtral": OllamaProvider(model="mixtral"),
    }
)

# User profile — controls crisis resources + system prompt tone
config = SafetyRouterConfig(
    user_name="Alex",
    user_age_range="Under 18",   # → youth-aware system prompts
    user_country="AU",            # → Lifeline AU (13 11 14) for crisis
    self_harm_threshold=0.70,     # emergency cutoff
    helpline_threshold=0.60,      # helpline cutoff
    custom_routing={              # override any category
        "gender":   "claude",
        "religion": "gemini",
    },
)

router = SafetyRouter(config=config)

Features

Built for responsible AI

Every feature is designed around the principle that safety checks should never be an afterthought.

✏️
Bias rephrasing
Every response includes a rephrased_text block — the original rewritten to remove bias with a changelog, meaning-preservation flag, and change-risk indicator. Runs locally at zero cost.
🧠
Mental health detection
Four risk signals scored on every request: self_harm, severe_distress, existential_crisis, emotional_dependency. All local, configurable thresholds.
🚨
Two-tier crisis escalation
Emergency tier skips the LLM entirely — even in streaming mode. Helpline tier appends support info. Crisis checks fire before the first token. 15 countries built in.
⚡
Zero-cost classification
All classification runs on your machine via Ollama (gemma3n:e2b). No API calls, no cost, no data leaving your environment until you choose to route.
🎯
Benchmark-backed routing
Each bias category routes to the model with the highest accuracy from a 270-sample benchmark. Override any mapping with SR_CUSTOM_ROUTING.
🌊
Safe streaming
Token streaming with full escalation checks — emergency and helpline logic fires before any tokens are yielded. No unsafe responses leak through the stream.
🔌
Pluggable providers
OpenAI, Anthropic, Google, Groq, and Ollama out of the box. Bring your own by subclassing BaseProvider. Providers are lazily loaded — only installed extras are imported.
🏠
Fully local mode
Route everything to local Ollama models. Zero external API dependency — perfect for air-gapped environments, local dev, or privacy-sensitive workloads.
🛡️
Production hardened v0.2.3
Rate limiting (60 req/min), input length validation, classifier fallback on malformed JSON, provider error isolation, thread-safe server init, 26 unit tests.
Configuration

All environment variables

Every option can be set via environment variable or passed directly to SafetyRouterConfig. safetyrouter setup writes a .env file automatically.

Variable Default Description
SR_CLASSIFIER_MODEL gemma3n:e2b Ollama model used for local bias + mental health classification
SR_USER_NAME — User's name — used in crisis transcript and age-aware system prompts
SR_USER_AGE_RANGE — Age range (Under 18, 18–25, …, 60+) — activates youth/elder-aware prompts
SR_USER_COUNTRY US ISO-2 country code or full name — determines crisis helpline and emergency number
SR_SELF_HARM_THRESHOLD 0.70 self_harm score ≥ this triggers Tier 1 emergency (LLM skipped)
SR_HELPLINE_THRESHOLD 0.60 severe_distress / existential_crisis ≥ this triggers Tier 2 helpline append
SR_CUSTOM_ROUTING {} JSON map of bias category → provider override, e.g. {"gender":"claude"}
OPENAI_API_KEY — Required for GPT-4 routing (gender, disability, religion). Install: pip install "safetyrouter[openai]"
ANTHROPIC_API_KEY — Required for Claude routing (race, age, sexual_orientation, socioeconomic_status)
GOOGLE_API_KEY — Required for Gemini routing (nationality, physical_appearance)
GROQ_API_KEY — Optional — Groq/Mixtral as a fast fallback or custom routing target
Programmatic configuration
python 
from safetyrouter import SafetyRouter, SafetyRouterConfig

config = SafetyRouterConfig(
    classifier_model="gemma3n:e2b",
    user_name="Alex",
    user_age_range="18–25",
    user_country="UK",            # → Samaritans 116 123 for crisis
    self_harm_threshold=0.70,
    helpline_threshold=0.60,
    custom_routing={"gender": "gemini"},
)
router = SafetyRouter(config=config)
